# DeFi Real Estate

## Prerequisites

Before you begin, ensure your environment meets the following requirements:

1. **Node.js** (version **18.x** or higher) [[Download](https://nodejs.org/)]
2. **Git** [[Download](https://git-scm.com/downloads)]
3. **Code Editor** (optional) [[VS Code](https://code.visualstudio.com/)]

## Getting Started

### 1. Clone the Repository

```bash
git clone <repository-url>
cd <repository-folder>
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Running the Application

```bash
npm run dev
```

Once the server is running, open your web browser and navigate to: `http://localhost:3000/`

## Troubleshooting

### - Check Node.js version

```bash
node -v
```

### - Clear Dependencies

```bash
rm -rf node_modules package-lock.json
```

### - Clean npm Cache

```bash
npm cache clean --force
```

### - Reinstall Dependencies

```bash
npm install
```
